# Sean Watson — Web Design Portfolio

Personal web design portfolio site. Static, single-page, fully self-contained HTML.

## Pages
- `index.html` — main portfolio (Web Studio)
- `logo.html` — SW logo page

## Hosting
Deployed via [Cloudflare Pages](https://pages.cloudflare.com/), connected to this GitHub repo. Every push to `main` triggers an automatic deploy.

No build step — Cloudflare serves the static files directly.
